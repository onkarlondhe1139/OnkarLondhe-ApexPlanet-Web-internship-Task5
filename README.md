## About The Project

This project is designed as a professional portfolio for a Data Analyst and Data Science enthusiast. It highlights practical experience with Python, SQL, Excel, Power BI, dashboard development, data cleaning, exploratory data analysis, KPI tracking, and insight communication.

The app is built with core web technologies:

- HTML5
- CSS3
- JavaScript
- Node.js static server

---

## Features

- Responsive portfolio layout for desktop, tablet, and mobile screens
- Resume-based hero section and professional profile photo
- Dynamic project browser with search, filter, sort, and save functionality
- Data analytics project cards for Power BI, Excel, and Python projects
- Internship timeline with detailed experience
- Skills, soft skills, education, and certifications sections
- Resume download link
- Linkedin and GitHub buttons
- Contact form with front-end validation
- Theme toggle with local storage persistence
- Optimized assets using minified CSS and JavaScript

## Run Locally

Open the project folder.

Install dependencies if required:

```bash
npm install
```

Start the local server:

```bash
npm start
```

Open the app in your browser:

```bash
http://localhost:4000
```

## Deploy On Netlify

### Quick Deploy

1. Go to Netlify Drop
2. Drag and drop the `public` folder
3. Netlify will generate a live website URL

### GitHub Deploy

1. Push this project to a GitHub repository
2. Open Netlify and select **Add new project**
3. Import the GitHub repository
4. Use these settings:

```bash
Build command: leave empty
Publish directory: public
```

5. Deploy the site
